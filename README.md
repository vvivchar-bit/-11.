# Практичне заняття №11 — Кореляційний аналіз

**Дисципліна:** Теорія ймовірностей та математична статистика

**Група:** АІК-43

## Мета роботи

Навчитись визначати тип та тісноту взаємозв’язку між випадковими змінними,
обирати метод кореляційного аналізу та робити висновок за отриманим
коефіцієнтом кореляції.

## Використані інструменти

- PSPP
- CSV-дані
- Кореляційний аналіз
- Критерій Колмогорова-Смірнова
- Коефіцієнт рангової кореляції Спірмена

## Хід виконання

1. Дані було імпортовано у PSPP з CSV-файлу.
2. Побудовано діаграму розсіювання для змінних `Lipoproteins` та `Hemoglobin`.
3. Перевірено нормальність розподілу за критерієм Колмогорова-Смірнова.
4. Оскільки нормальність розподілу не підтвердилась, для аналізу використано
   коефіцієнт рангової кореляції Спірмена.
5. У PSPP коефіцієнт Спірмена було відтворено через кореляцію між ранговими
   змінними `R_Lipoproteins` та `R_Hemoglobin`.

## Результат

Отримано:

- коефіцієнт кореляції: **r = -0.192**;
- значущість: **p = 0.000**;
- кількість парних спостережень: **N = 389**.

## Висновок

Між змінними `Lipoproteins` та `Hemoglobin` виявлено слабкий зворотний
статистично значущий зв’язок. Оскільки коефіцієнт кореляції має від’ємне
значення, зі зростанням однієї змінної інша має слабку тенденцію до зменшення.
